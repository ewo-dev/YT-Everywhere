# YT Everywhere

**YT Everywhere** is a Firefox extension that lets you search YouTube instantly without leaving the page you're currently browsing.

With a simple keyboard shortcut, a clean and minimal search bar appears in the center of your screen. Type your query, press **Enter**, and your search opens directly on YouTube.

The goal of YT Everywhere is to provide a fast, distraction-free experience inspired by tools like Spotlight and Raycast, making YouTube search available from anywhere on the web.

## Features

### Current

- Open the search bar with a keyboard shortcut
- Centered, minimal search interface
- Automatic focus on the search input
- Search YouTube by pressing **Enter**
- Close the overlay with **Escape**
- Close the overlay by clicking outside

### Planned

- Real-time search suggestions
- Search history
- Customizable keyboard shortcuts
- Reuse an existing YouTube tab when available
- Light and dark themes
- Settings page
- YouTube Music support
- Chrome and Chromium compatibility

## Tech Stack

- TypeScript

## Project Goals

- Clean and modular architecture
- Strongly typed code
- Easy to maintain and extend
- Smooth and responsive user experience
- Cross-browser compatibility

## Vision

YT Everywhere aims to become the fastest way to access YouTube from anywhere on the web.

Future versions may include smart search suggestions, quick actions, playlist shortcuts, and additional productivity features while keeping the experience lightweight, fast, and intuitive.
